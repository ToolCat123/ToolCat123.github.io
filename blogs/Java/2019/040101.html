# 搭建Java开发环境

# 安装Java并配置环境变量

  1. 下载并安装 JDK
  2. 创建系统变量 JAVA_HOME 值为JDK安装目录
  3. 在path中添加 %JAVA_HOME%\bin

# 安装maven并配置环境变量

  1. 下载并安装 maven
  2. 创建系统变量 MAVEN_HOME 值为maven安装目录
  3. 在path中添加 %MAVEN_HOME%\bin

# 安装IntelliJ IDEA开发工具并导入设置

  1. 下载并安装 IntelliJ IDEA
  2. 导入自己idea的settings文件
  3. 安装常用插件(Lombok,MapStruct support,Free Mybatis plugin,RESTfulToolkit,Vue.js,Codota)
